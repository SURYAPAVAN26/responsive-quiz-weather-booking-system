# Responsive Quiz, Image Carousel, Weather & Booking System

## Overview

This project is a responsive web application developed using HTML, CSS, and JavaScript. It demonstrates frontend development concepts such as responsive design, DOM manipulation, event handling, API integration, and form handling.

## Features

### Responsive Design
- Mobile-friendly layout using CSS Media Queries.
- Adapts to different screen sizes including mobile, tablet, and desktop devices.

### Interactive Quiz
- Displays a multiple-choice question.
- Validates user answers instantly.
- Shows correct or incorrect feedback.

### Image Carousel
- Allows users to browse through multiple images.
- Supports Next and Previous navigation.

### Weather API Integration
- Fetches real-time weather data using the Open-Meteo API.
- Displays current temperature information.
- Uses JavaScript Fetch API and JSON processing.

### Booking System
- Users can enter their name, date, and time.
- Validates form inputs.
- Displays booking confirmation details.

## Technologies Used

- HTML5
- CSS3
- JavaScript (ES6)
- Fetch API
- Open-Meteo Weather API
